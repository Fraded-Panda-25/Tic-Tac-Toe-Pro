# 🎮 Tic-Tac-Toe Pro

![Tic Tac Toe Pro Banner](https://copilot.microsoft.com/th/id/BCO.207b3084-4e1c-47ce-8356-a3df6f220a41.png)

A modern, feature-rich implementation of the classic Tic-Tac-Toe game — built for fun, learning, and showcasing advanced web development techniques.

---

## ✨ Features

- **Multiple Board Sizes**: Play on 3x3, 4x4, or 5x5 grids.
- **Game Modes**:
  - Local multiplayer
  - AI opponents with difficulty levels (Casual & Impossible using Minimax + Alpha-Beta Pruning)
- **Smart UX**:
  - Coin toss system to decide first turn
  - Turn indicators and game state persistence
  - Clean, responsive design
- **Customization**:
  - Player names
  - Theme options
- **Performance**:
  - Optimized algorithms for smooth gameplay
  - Lightweight and fast

---

## 🛠️ Tech Stack

- **HTML5** — Structure
- **CSS3** — Styling & responsive design
- **JavaScript (ES6+)** — Game logic, AI, and interactivity
